05 — Design prompt

# SOFKINO — Brand Identity

Concept:  Qorong'i kinozal + iliq proyektor nuri (amber/gold beam).
Meaning:  SOF = toza/sifatli, KINO = cinema → "sifatli kino".
Mood:     zamonaviy, premium, ishonchli, do'stona. Dark-first.

Logo:     oltin (#F6B24A→#E7902B) yumaloq kvadrat + qorong'i
          "play" uchburchagi (proyektor nuri). Wordmark:
          SOF (oltin gradient) + KINO (oq). Favicon, yumaloq
          avatar va mono variantlar.

Colors:   fon #0A0C14, karta #141926, chegara #262E42,
          urg'u #F6B24A, ikkilamchi indigo #7C8CFF,
          semantik: yashil #34D399, ko'k #60A5FA, qizil #F26D6D.

Type:     Inter / geometrik sans. Sarlavha 800, keng tracking.
          Lotin o'zbek alifbosi.

Deliver: logo to'plami (SVG+PNG), bot avatar 512×512,
          kanal banner 1500×500, janr ikonalari, admin UI kit,
          bot kino-kartochka shabloni. Dark-mode, radius 14–20px,
          yumshoq soya, cinematic glow (me'yorida).
